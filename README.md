# DC Transportes

Site institucional responsivo para apresentar a DC Transportes com uma experiencia visual direta, moderna e focada em conversao.

## Destaques

- Hero com identidade visual da transportadora.
- Secoes para servicos, diferenciais e contato.
- Layout responsivo para desktop e celular.
- Estrutura simples com HTML, CSS e JavaScript puro.
